# bcbk2023-web

Barcamp Bangkhen 2023 Webpage

## Getting Started

### Prerequisites

| Name | Version |
| ---- | ------- |
| npm  | any     |
| Node | 16.13.2 |

### Installation

1. Create an `.env` at the root directory that contains

```
PORT=
MONGO_URI=
```

2. Run locally by using Docker

```
docker compose build
docker compose up -d
```
